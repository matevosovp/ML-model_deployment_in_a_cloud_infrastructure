# Проект: релиз модели оценки стоимости недвижимости в продакшен

Добро пожаловать в репозиторий проекта 3-го спринта Практикума. Цель проекта — вывести готовую ML-модель для оценки цен на недвижимость в продакшен и обеспечить онлайн-доступ к предсказаниям, гибкое развертывание и мониторинг.

Полное описание задания находится в уроке «Проект. Релиз модели в продакшен» на учебной платформе.

## Бизнес-контекст
Мы работаем в Яндекс Недвижимости. Команде требуется сервис, который позволяет коллегам получать предсказания цены объекта недвижимости в режиме онлайн, а также обеспечивает наблюдаемость и контроль рисков в эксплуатации.

## Что реализовано в проекте
- FastAPI-микросервис для выдачи предсказаний модели по HTTP
- Контейнеризация микросервиса с помощью Docker
- Запуск микросервиса и системы мониторинга через Docker Compose
- Сбор и визуализация метрик с помощью Prometheus и Grafana
- Дашборд Grafana и скрипт для симуляции нагрузки на сервис

## Технологии
- Python
- FastAPI, Uvicorn
- Docker, Docker Compose
- Prometheus
- Grafana
- prometheus_client, prometheus_fastapi_instrumentator

## Структура репозитория
- `services/ml_service/` — код FastAPI-микросервиса
- `services/models/` — файл модели для предсказаний
- `services/prometheus/` — конфигурация Prometheus
- `services/.env` — переменные окружения (для сдачи проекта хранится в репозитории)
- `Dockerfile_ml_service` — Dockerfile для сборки образа микросервиса
- `docker-compose.yaml` — запуск микросервиса и мониторинга в Docker Compose
- `dashboard.json` — экспорт дашборда Grafana
- `dashboard.jpg` — скриншот дашборда Grafana
- `Instructions.md` — пошаговые инструкции по запуску этапов проекта
- `Monitoring.md` — описание метрик и логика построения дашборда
